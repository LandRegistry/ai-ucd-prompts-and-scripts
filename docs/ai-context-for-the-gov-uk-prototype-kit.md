## What this does

A markdown file to help AI assistants generate correct, accessible, GOV.UK-compliant code for prototypes. 

If this file is provided to an IDE using an Ai assitant, it will follow the rules.  Attribution to the creator of these rules are in the file. 

If you are not using an IDE with Ai assistance, this file is worth reading, as it contains a lot of rules that could be used in other ways and a lot of inspiration for other things you could do for prototyping or checking quality of built services - at all stages of design and development.

I've not used this file in an IDE with Ai assistant.  I've used chunks of the rules in the file for Ai promting and agents. For example to check coding rules have been followed.
